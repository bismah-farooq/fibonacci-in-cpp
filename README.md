# Fibonacci in C++

This project demonstrates how to generate the Fibonacci sequence using C++.  
It is a simple console-based program for learning and practice.

## Description
The Fibonacci sequence is a series of numbers where each number is the sum of the two preceding ones:

0, 1, 1, 2, 3, 5, 8, ...

This program calculates and displays Fibonacci numbers in the terminal.

## Technologies Used
- C++
- Xcode
- macOS Terminal
- Linux

## How to Run
1. Compile the program:
```text
   g++ main.cpp -o fibonacci
```
2. Run the program:
```text
   ./fibonacci
```
